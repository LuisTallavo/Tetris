# Tetris

A classic Tetris game built with Python and Pygame, playable in the browser.

## Play Online

[Play the game here]([https://ltallavo.github.io/Tetris/](https://luistallavo.github.io/Tetris/))

## Run Locally

**In Browser:**
```bash
pip install pygbag
python run_browser.py
```

**Desktop:**
```bash
pip install pygame
python main.py
```

## Controls

- **Arrow Left/Right** - Move piece
- **Arrow Up** - Rotate clockwise
- **Arrow Down** - Rotate counter-clockwise
- **Space** - Instant drop
- **D** - Move down
- **T** - Use slowtime power-up
- **S** - Swap piece

## Tech Stack

- Python
- Pygame
- Pygbag (WebAssembly compilation for browser)
- GitHub Actions (automated deployment)
